# Security-Based Manual Testing Project

## Project Overview
This project demonstrates manual security testing performed on an Online Banking Web Application. The goal of the project is to identify security vulnerabilities, validate authentication and authorization mechanisms, and ensure secure application behavior.

The project includes complete QA documentation commonly used in real-time software testing environments.
